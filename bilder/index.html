Original Bild

Bei diesem Bild handelt es sich um das Originalbild ohne jegliche Bearbeitung. Dieses Bild lässt sich hier finden. Scrolle weiter runter, um einige Filter zu testen.

Grayscale

Dieser Filter wird genutzt um die Sättigung zu einem bestimmten Prozentsatz aus dem Bild zu entfernen.

filter: grayscale(

%);

Contrast

Mit diesem Filter wird der Kontrast erhöht oder verringert.

filter: contrast(

%);

Hue-Rotate

Mit diesem Filter lassen sich alle Farben auf dem Bild rotieren.

filter: grayscale(

deg);

Invert

Invertiert die Ursprungsfarben des Bildes.

filter: invert(

%);

Opacity

Kontrolliere die Transparenz des Bildes.

filter: opacity(

%);

Unvollständige Liste. Alle filter Eigenschaften können bei den MDN Web Docs von moz://a gefunden werden.
