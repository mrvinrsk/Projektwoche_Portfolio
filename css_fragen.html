Was versteht man bei CSS unter "Vererbung"?

Der Wert 'inherit' sorgt dafür, dass der Wert des Elternelements für das Kindelement übernommen wird.
Hat beispielsweise ein <div> die Eigenschaft color: #EEE; und ein Text innerhalb dieses divs color: inherit;, dann wird dieser Text ebenfalls in #EEE angezeigt.

Welchen Vorteil haben externe CSS-Dateien?

Eine externe CSS Datei sorgt für mehr Übersichtlichkeit und kann auf mehreren Seiten eingebunden werden, sodass man nicht immer wieder gleiche Regeln schreiben muss.
